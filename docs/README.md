## What is TOON?

TOON (Token-Oriented Object Notation) is a data format designed for Large Language Model applications. It provides:

- **Token Efficiency**: 30-60% fewer tokens than JSON
- **Structure Clarity**: Explicit array lengths and field headers
- **Human Readability**: Familiar syntax patterns
- **LLM Optimization**: Better parsing and generation for AI models

## Extension Features

This VS Code extension provides complete language support for TOON files:

- Syntax highlighting with color coding
- IntelliSense with smart autocomplete
- Real-time validation and error checking
- Auto-formatting with configurable options
- Code snippets for common patterns
- Bidirectional JSON conversion
- Hover documentation and help
