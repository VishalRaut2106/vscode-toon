## ğŸ’¡ Key Concepts

### What Are Snippets?
Code templates that help you quickly insert common TOON patterns. Type a prefix and press Tab to insert a complete template with placeholders.

### Why Use Snippets?
- **Save time** - Insert templates instead of typing
- **Reduce errors** - Templates ensure correct syntax
- **Learn patterns** - See best practices in action
- **Improve efficiency** - Write code faster

### How Do They Work?
1. Type snippet prefix (e.g., `toon-array-tabular`)
2. Press Tab to insert template
3. Edit placeholders
4. Press Tab to navigate between placeholders
5. Done!

### Most Common Snippet
**Tabular Array** (`toon-array-tabular`) - Use for 80% of your code
```toon
items[2]{id,name}:
  1,Alice
  2,Bob
```
